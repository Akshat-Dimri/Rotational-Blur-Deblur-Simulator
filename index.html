Rotational Deblurring Pipeline Simulator
Hello! This tool simulates and corrects rotational motion blur — the kind that occurs when an airborne camera rotates during image capture.

What it does: Given an image, it synthetically blurs it using configurable angular velocity and exposure time, then applies one of seven deblurring algorithms (Inverse, Wiener, Tikhonov, Richardson-Lucy, Total Variation, Kalman, Blind Deconvolution) and reports quality metrics like PSNR, SSIM, and MSE.

How to use: Upload an image → adjust blur parameters → pick a filter → click Run deblur. Use Compare all filters to benchmark them side by side.
